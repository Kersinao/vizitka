# Моя визитка и проекты

## Что умею прямо сейчас

- **Python + Pygame**: «Бегущий квадрат» — полноценный игровой цикл, обработка ввода, коллизии через `pygame.Rect`, счёт, отрисовка, контроль FPS.
- **HTML/CSS**: сделал сайт‑визитку — базовая вёрстка, адаптивность, понятная структура страницы.
- **Git**: инициализация репозитория, коммиты с понятными сообщениями, пуш на GitHub, решение частых ошибок.

## Над чем работаю и куда двигаюсь (под задачи инфраструктурных проектов)

Ориентируюсь на бэкенд‑ и инфраструктурные задачи уровня Kaspersky Secure Mail Gateway:

- **Linux**: осваиваю терминал и базовые команды, учусь запускать и диагностировать сервисы, работать с виртуальными окружениями.
- **Сети и протоколы**: разбираю стек TCP/IP, чтобы понимать, как устроены сетевые взаимодействия; планирую добавить в пет‑проекты простые HTTP‑запросы и работу с REST.
- **Базы данных**: изучаю SQL и планирую внедрить сохранение данных (например, рекордов) через PostgreSQL.
- **Асинхронность и параллелизм**: разбираюсь в различиях потоков, процессов и `asyncio`, чтобы лучше понимать многопоточные задачи.
- **CI/CD**: тренирую чистую историю коммитов как базу для интеграции с CI/CD.
- **Тестирование**: начинаю писать unit‑тесты для отдельных модулей, чтобы код был надёжнее.
